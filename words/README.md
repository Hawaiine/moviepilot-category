# words — 自定义识别词库

`common-words.txt` 是一个共享词库，用于解决资源命名不规范导致的识别错误。

## 导入方式

在 MP WebUI → **设定 → 词表** 中，粘贴 `common-words.txt` 内容即可。

## 词库格式

```
# 1. 屏蔽词（单独一行，从资源名中删除）
CMCT
HDSWEB

# 2. 替换规则（被替换词 => 替换词）
BluRay => Blu-ray
WEB-DL => WEBDL

# 3. 集数定位（前定位 <> 后定位 >> 偏移）
第<>话 >> 0
第<>集 >> 0
```

## 规则类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 屏蔽词 | `CMCT` | 从识别结果中删除 |
| 替换 | `BluRay => Blu-ray` | 用替换词替代被替换词 |
| 集数定位 | `第<>话 >> 0` | 定位集数位置，`>>` 后填偏移量 |
| 组合 | `第<>集EP >> 0` | 替换 + 定位 + 偏移 |
